# Flappy-Bird

This is a mini-project by Rex Uriel I. Villaflores. It is a simple recreation of the game Flappy Bird using Java. The "Best Score" feature is not implemented :'(.

# How to run

Since the program is in Java, the .java files must be compiled before running. To compile the files, run these commands in your terminal:

```
javac App.java
javac FlappyBird.java
```

After compilation, you may run the game using this command:

```
java App.java
```

Background, Flappy bird sprite, and both top and bottom pipe sprites are taken from Kenny Yip Coding.

Github source: https://github.com/ImKennyYip/flappy-bird-java
